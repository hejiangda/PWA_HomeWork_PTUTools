# P图小工具
## 使用本工具可以方便地为您的图片添加滤镜，改变风格。

### 开发环境：

+ Ubuntu 16.04
+ Visual Studio Code 1.38.1
+ opencv.js 3.4
+ bootstrap 4.3.1
